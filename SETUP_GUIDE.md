# オーディションダッシュボード セットアップガイド

ダッシュボードの「＋追加」ボタンからスプレッドシートに直接データを書き込むためのGAS WebApp設定手順です。

---

## 全体の流れ

```
① スプレッドシートの「ウェブに公開」設定（データ読み込み用）
② GASプロジェクト作成 & スクリプト貼り付け
③ GASをウェブアプリとしてデプロイ
④ 発行されたURLをダッシュボードに設定
```

---

## ① スプレッドシートの「ウェブに公開」設定

> これはダッシュボードがスプレッドシートのデータを**読み込む**ための設定です。

1. [GATE_オーディション管理スプレッドシート](https://docs.google.com/spreadsheets/d/1NBZce4Xqx7FgbXujQlaO15xd-nAZvUmAqBVGRBgIaGU/edit) を開く
2. メニュー → **ファイル** → **共有** → **ウェブに公開**
3. ダイアログが出たら以下を設定：
   - ドキュメント全体 / ウェブページ → そのままでOK
   - **「公開」ボタンを押す**
4. 確認ダイアログ → **OK** をクリック

> [!IMPORTANT]
> これをやらないとダッシュボードはフォールバックデータ（サンプル）で表示されます。

---

## ② GASプロジェクト作成 & スクリプト貼り付け

### 方法A: スプレッドシートから作成（おすすめ）

1. [GATE_オーディション管理スプレッドシート](https://docs.google.com/spreadsheets/d/1NBZce4Xqx7FgbXujQlaO15xd-nAZvUmAqBVGRBgIaGU/edit) を開く
2. メニュー → **拡張機能** → **Apps Script**
3. 新しいタブでGASエディタが開く
4. 既存のコード（`function myFunction() {}` など）を**すべて削除**
5. 以下のファイルの中身を**すべてコピーして貼り付け**：

```
📄 gas_webapp_api.js （このフォルダ内のファイル）
```

> ファイルの場所: `社内/audition_dashboard/gas_webapp_api.js`

6. **Ctrl + S** で保存（プロジェクト名は「GATE_オーディションAPI」など）

### 方法B: 直接GASプロジェクト作成

1. https://script.google.com/ にアクセス
2. **「新しいプロジェクト」** をクリック
3. 上記と同じようにコードを貼り付け

---

## ③ GASをウェブアプリとしてデプロイ

1. GASエディタ右上の **「デプロイ」** ボタン → **「新しいデプロイ」**
2. 左のギアアイコン⚙️をクリック → **「ウェブアプリ」** を選択
3. 以下の設定にする：

| 項目 | 設定値 |
|------|--------|
| 説明 | `GATE オーディション追加API` |
| 次のユーザーとして実行 | **自分** |
| アクセスできるユーザー | **全員** |

4. **「デプロイ」** ボタンをクリック
5. **初回のみ**: 「アクセスを承認」が出るので：
   - **「アクセスを確認」** をクリック
   - Googleアカウントを選択
   - 「このアプリは確認されていません」→ **「詳細」** → **「〇〇に移動（安全ではないページ）」**
   - **「許可」** をクリック
6. **ウェブアプリのURLが表示される** → これをコピー！

```
例: https://script.google.com/macros/s/AKfycbx.../exec
```

> [!WARNING]
> このURLは外部公開されるため、取り扱いに注意してください。URLを知っている人は誰でもスプレッドシートにデータを追加できます。

---

## ④ ダッシュボードにURLを設定

1. `社内/audition_dashboard/app.js` を開く
2. 以下の行を見つける（ファイル後半にあります）：

```javascript
const GAS_WEBAPP_URL = '';
```

3. コピーしたURLを貼り付ける：

```javascript
const GAS_WEBAPP_URL = 'https://script.google.com/macros/s/AKfycbx.../exec';
```

4. 保存して完了！

---

## 動作確認

1. ダッシュボードを開く（`index.html`）
2. 右下の赤い **＋ボタン** をクリック
3. フォームに入力：
   - タレント名: 選択
   - ジャンル: 選択
   - オーディション名: 入力
   - （任意）ファイルをドラッグ&ドロップ
4. **「追加する」** をクリック
5. 成功すると **「✅ スプレッドシートに追加しました！」** のトーストが出る
6. スプレッドシートを確認 → 最終行にデータが追加されている

---

## 資料ファイルの保存先

添付されたファイル（AD、台本、写真など）は自動的にこのGoogleドライブフォルダに保存されます：

📁 [GATE_オーディション資料](https://drive.google.com/drive/folders/1EyDWtQ_yBsGMpsw0ZKGsImFgVHjQOAhd)

保存されたファイルのURLは、スプレッドシートの「資料リンク」列（N列）に自動で記録されます。

## トラブルシューティング

### ❌ 「データ読み込みエラー」が出る
→ ①の「ウェブに公開」設定を確認

### ❌ 「追加する」を押してもスプシに反映されない
→ ④のGAS_WEBAPP_URLが正しく設定されているか確認
→ ブラウザのDevToolsコンソール(F12)でエラーを確認

### ❌ 「このアプリは確認されていません」で止まる
→ 「詳細」→「〇〇に移動（安全ではないページ）」をクリック（自分のスクリプトなので安全です）

### ❌ GASを更新した場合
→ 「デプロイ」→「デプロイを管理」→「✏️ 編集」→バージョンを「新しいバージョン」に変更 → 「デプロイ」
→ **URLは変わりません**

---

## 構成図

```
[ダッシュボード (HTML)]
    │
    ├── GET → Google Sheets Visualization API → データ読み込み
    │         （ウェブに公開 必須）
    │
    └── POST → GAS WebApp (doPost) ─┬── スプレッドシートに行追加
                                     └── ファイル → Google Drive に保存
                                              ↓
                                         資料リンク列に URL 記録
```
